# Responsibilities

I am a full-stack developer on this project, handling both frontend and backend development across the Korean ecommerce platform.

## Key Responsibilities

- **Feature Development** — implementing new features for the Korean version of the ecommerce platform, covering the user-facing storefront (NextJS), the admin panel, and the backend service (Spring Boot)

- **Bug Fixing & Maintenance** — identifying and resolving bugs across the full stack, including frontend rendering issues, backend API errors, and database inconsistencies

- **AI Rendering Engine Integration** — working closely with the team responsible for the AI rendering engine to ensure smooth integration, including handling API communication, error handling, and performance optimization

- **AI Credit Management**: implementing a system to track and manage AI credit usage for users, ensuring that the platform can monitor and control costs associated with AI rendering requests

- **Database Management** — working with MariaDB for data modeling, query optimization, and schema changes to support new features

## Day-to-Day Activities

- Participate in daily stand-up meetings with the Korean client's team
- Write clean, maintainable code following the team's coding standards
- Conduct and participate in code reviews via Bitbucket pull requests
- Coordinate with the client's product team on feature requirements and timelines
- Troubleshoot production issues on the EC2-hosted environment

## Tips for Success

### 1. Time Management
- Use a timer (Pomodoro: 50 min work + 10 min break)
- Don't get stuck >15 min on one question; mark and move on
- Track actual time spent vs. planned

### 2. Active Learning
- Write code by hand first, then type
- Explain your solution out loud
- Create summary notes for each topic

### 3. Energy Management
- Sleep 7-8 hours nightly (non-negotiable)
- Stay hydrated (2-3L water/day)
- Eat proper meals, avoid heavy carbs at lunch
- Take real breaks away from the computer

### 4. When Stuck
1. Read the question again carefully
2. Work through a small example by hand
3. Think about edge cases
4. If still stuck after 15 min, check solution, understand, then re-solve

### 5. Priority Order (if running behind)
1. ✅ SQL Practice (most relevant to your interview)
2. ✅ DB Design (directly applicable)
3. ✅ Write a Program (coding fundamentals)
4. ✅ Coding Interview Easy (supplementary)

**3.** Describe a typical day in your current/last project. What tools do you use daily?

========== THIS SECTION IS THE ANSWER ==========

Morning starts with a 15-minute daily standup — what I did yesterday, what I'm doing today, any blockers. Then I check Slack for any overnight issues (we had users in US timezone) and review any open PRs from teammates.

Most of my day is coding in IntelliJ IDEA — writing Java/Spring Boot code, writing tests, and debugging. I use Git with GitHub for version control, create PRs with mandatory code review. For deployment, GitHub Actions handles CI and we deploy via Elastic Beanstalk. When I need to debug production issues, I use Kibana for logs and New Relic for APM and tracing. For communication, Slack and Jira daily.

Tools I use daily: IntelliJ, Git/GitHub, Jira, Slack, Kibana, New Relic, Postman (for API testing), DBeaver (for database queries), Docker (local development). Occasionally: AWS Console, SonarCloud for code quality.

**1.** Tell me about yourself and walk me through your most recent project.

========== THIS SECTION IS THE ANSWER ==========

I have over 5 years of experience as a backend developer, primarily with Java and Spring Boot. My most recent project was a vehicle marketplace — one of the largest automotive sites in the US with around 30 million user visits per month. I was responsible for the Vehicle Search Results Page, both frontend and backend.

On the backend, we used Spring Boot in a microservices architecture. The main search service was an aggregation service that talked to Apache Solr for search, plus several satellite services backed by PostgreSQL and DynamoDB. The system was deployed across two AWS regions (us-east-1, us-west-2) for high availability, heavily integrated with AWS services like Lambda, SQS, and SNS.

On the frontend side, I also worked with Next.js and a custom Node.js framework. I was involved in key initiatives like decoupling the monolith into microservices, building a natural language search feature with AWS Bedrock, SEO optimization, and developing an accessible UI library for users with visual impairments. Beyond coding, I participated in code reviews, daily stand-ups, and deployment processes using GitHub Actions and Elastic Beanstalk.

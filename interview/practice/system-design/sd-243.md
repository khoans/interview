**243.** What are well-known ports? Name 10 important ones (22, 80, 443, 3306, 5432, 6379, etc.).

Well-known ports (0-1023) are reserved for standard services by IANA. Key ports: **22** (SSH), **80** (HTTP), **443** (HTTPS), **3306** (MySQL), **5432** (PostgreSQL), **6379** (Redis), **8080** (HTTP alternate/Tomcat), **8443** (HTTPS alternate), **27017** (MongoDB), **9092** (Kafka), **8983** (Apache Solr), **53** (DNS), **25** (SMTP).

On our automotive marketplace, our security groups were configured around these ports: ALB listened on 443 (HTTPS via CloudFront), Spring Boot services ran on 8080 inside ECS containers, PostgreSQL on 5432 (only accessible from application security group), ElastiCache Redis on 6379 (only from application SG), and Apache Solr on 8983 for our vehicle search index. We never exposed database or cache ports to the internet -- only the ALB on 443 was publicly accessible.

The lesson: never change default ports as a "security" measure (security through obscurity). Instead, use proper security groups and network ACLs to restrict access. A port scan finds non-standard ports in seconds.
